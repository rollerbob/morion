Поиск параметров команды во входящем буфере. Ищуться все вхождения разделённые "%", затем эти вхождения проверяются на валидность и сохраняются внутри объекта команды.

Parameters
buffбуфер в котором происходит поиск.
buff_sizeразмер буфера.
cmdуказатель на команду, в которую сохраняются параметры
amountсколько групп параметров искать.
Returns
uint32_t статус поиска. 0 - ошибка во входных параметрах, 1 - входные параметры успешно установлены в принятой команде.
